## .개요
네이버 검색 API를 활용한 테스트 코드입니다!!

## 실행 환경
node, react, express

## 실행 순서
1. package 설치
   ```
   npm install
   ```
2. express mongo 서버 실행
   ```
   node server.js
   ```
3. front 실행
   ```
   npm run start
   ```
