## 실행 환경
node, react, express

## 실행 순서
### 1. 네이버
1-1. package 설치
   ```
   npm install
   ```
1-2. express mongo 서버 실행
   ```
   node server.js
   ```
1-3. front 실행
   ```
   npm run start
   ```

### 2. 구글
   ```
   npm start
   ```
